# Coctail Recipe

## Приложение для поиска коктелей

- Все коктели из базы TheCocktailDB

- Поиск по названию 

- Описание/рецепт/ингридиенты для каждого коктеля

- Возможность сохранить коктель в избранное (soon)


## Использовались

- Jetpack Compose
- Navigation
- MVVM
- Coroutines
- Dagger - Hilt
- Retrofit
- Coil
